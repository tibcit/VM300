# homepage

## Layout
- `appdata/` : données persistées (volumes montés par docker-compose).
- `config/` : fichiers de configuration versionnés.
- `logs/` : emplacement pour exporter les journaux si nécessaire.
- `secrets/` : fichiers utilisés comme Docker secrets.

## Compose
Consultez `docker-compose.yml` pour les montages et variables spécifiques au service.
